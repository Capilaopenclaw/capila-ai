## Časť 1: Finance & Accounting (10 Use Cases)

### Use Case 1: Automatizácia Konsolidácie
**Problém:** Investičný fond s 18 SPVs stráca 5 dní mesačne na manuálnu konsolidáciu.

**AI Riešenie:**
- Automatická extrakcia dát z účtovných systémov
- Mapovanie chart of accounts
- Generovanie konsolidovaných reportov

**Implementácia:**
1. Integrácia s existujúcimi ERP systémami
2. Nastavenie mapping rules
3. Automatizovaný reporting pipeline

**ROI:**
- Časová úspora: 40 hodín mesačne
- Finančná úspora: €15,000/rok (náklady na manuálnu prácu)
- Zníženie chýb: 95%

**Riziká & GDPR:**
- Dáta zostávajú vo vašej infraštruktúre
- Audit trail pre všetky transformácie
- Šifrovanie pri prenose

---

### Use Case 2: Predikcia Cash Flow
**Problém:** Neistota v cash flow plánovaní vedie k zbytočným úverom.

**AI Riešenie:**
- Time-series forecasting model
- Integrácia s bankovými transakciami
- Scenario planning

**Implementácia:**
1. Historická analýza 24-36 mesiacov
2. Feature engineering (sezónnosť, fakturačné cykly)
3. Model deployment a monitoring

**ROI:**
- Presnosť predikcie: ±5% (vs. ±20% manuálne)
- Optimalizácia working capital: €50,000-200,000
- Zníženie úrokových nákladov: 15-30%

---

### Use Case 3: Automatizované Účtovanie Faktúr
**Problém:** Manuálne zadávanie faktúr — 15 minút na dokument, 500 faktúr mesačne.

**AI Riešenie:**
- OCR + NLP extrakcia dát
- Automatické párovanie s PO
- GL code predikcia

**Implementácia:**
1. OCR pipeline (Azure Form Recognizer / Google Document AI)
2. ML model pre GL classification
3. ERP integrácia (SAP, Oracle, Dynamics)

**ROI:**
- Časová úspora: 120 hodín mesačne
- Presnosť: 98%+ (vs. 85% manuálne)
- Návratnosť investície: 3-6 mesiacov

---

### Use Case 4: Fraud Detection
**Problém:** Podvodné transakcie sú odhalené až po škode.

**AI Riešenie:**
- Anomaly detection na transakciách
- Real-time scoring
- Alerting system

**Implementácia:**
1. Historická analýza podvodov
2. Feature engineering (velocity, geolocation, device fingerprint)
3. Real-time scoring API

**ROI:**
- Detekcia: 90%+ podvodov pred realizáciou
- Finančná úspora: €100,000-500,000/rok
- Zníženie false positives: 70%

### Use Case 5: Automatizované Reporting

**AI Riešenie:**
- Automated data extraction
- Dynamic report generation
- Natural language summaries

**Implementácia:**
1. Data connectors (SQL, APIs, Excel)

### Use Case 6: Credit Risk Assessment
**Problém:** Manuálne hodnotenie bonity klientov — subjektívne, pomalé.

**AI Riešenie:**
- ML model pre credit scoring
- Alternative data (sociálne siete, transakcie)
- Explainable AI

**Implementácia:**
1. Data collection (interné + externé)
2. Model training (XGBoost / Neural Networks)
3. API pre real-time scoring

**ROI:**
- Rýchlosť: < 1 sekunda (vs. 2-3 dni)
- Presnosť: +15% oproti tradičným metódam
- Default rate: Zníženie o 20-30%
